# ReactJS Learning Repository

Welcome to your ReactJS learning journey! This repository contains practice materials and code examples to help you master React.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm
- A code editor (VS Code recommended)

### Installation

1. Initialize a new React project using Vite:
    ```bash
    npm create vite@latest my-react-app
    ```

2. Follow the prompts to select "React" as the framework.

3. Navigate and Install Dependencies:
    ```bash
    cd my-react-app
    npm i
    ```

4. Start the development server:
    ```bash
    npm run dev
    ```

## Learning Path

- **Basics**: JSX, Components, Props, State
- **Hooks**: useState, useEffect, useContext
- **Advanced**: Custom Hooks, Performance Optimization
- **Routing**: React Router
- **State Management**: Context API
