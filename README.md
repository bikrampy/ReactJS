# ReactJS Learning Repository

Welcome to your ReactJS learning journey! This repository contains practice materials and code examples to help you master React.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- A code editor (VS Code recommended)

### Installation

1. Clone or navigate to this repository
2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm start
    ```

## Learning Path

- **Basics**: JSX, Components, Props, State
- **Hooks**: useState, useEffect, useContext
- **Advanced**: Custom Hooks, Performance Optimization
- **Routing**: React Router
- **State Management**: Context API
